# panel-yihao-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面各自具备独立的功能或信息展示用途，不针对任何特定域名或具体网站。

## 项目简介

panel-yihao-guide 是一个静态 HTML 页面集合。仓库中的每个 HTML 文件均为独立页面，可直接在浏览器中打开使用。页面内容涵盖工具、信息展示、交互界面等多种类型，适合需要快速部署或分享的轻量级场景。

## 目录说明

```
.
├── index.html          # 主入口页面（如有）
├── page1.html          # 示例页面1
├── page2.html          # 示例页面2
├── assets/             # 资源文件夹（CSS、JS、图片等）
│   ├── style.css
│   ├── script.js
│   └── images/
└── README.md           # 本文件
```

- `index.html`：可选的主页入口，用于导航或汇总。
- 各 `.html` 文件：独立的 HTML 页面，文件名反映页面主题。
- `assets/`：存放页面引用的外部资源，包括样式表、脚本和图片。
